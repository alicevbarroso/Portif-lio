    Projetos

                Projeto: INSEG - Iniciativa Infância Segura.
            Descrição: Desenvolvi um site que permite aos usuários cadastrar crianças 
            desaparecidas e identifica-las. O site é intuitivo e foi projetado para ser 
            utilizado tanto em dispositivos móveis quanto em desktops, proporcionando uma 
            experiência consistente em diferentes plataformas.

            Tecnologias Utilizadas:

            Front-end: HTML, CSS, JavaScript
            Back-end: PHP
            Controle de Versão: Git/GitHub
            Destaques do Projeto:
            Responsividade: Layout responsivo, adaptando-se a diferentes tamanhos de tela, desde 
            smartphones até monitores widescreen.
            Meio de contato: A partir do formulário, o usuário tem a opção de adicionar sua criança 
            desaparecida ao site, após aprovação.

            Desafios e Soluções:
            Solução: Utilização de índices no MongoDB para otimizar as consultas e melhorar o
             desempenho do sistema.
            Link para o Repositório: GitHub - Sistema de Gestão de Tarefas
